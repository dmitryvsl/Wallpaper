# Wallpaper

Андроид приложение, с помощью которого можно найти красивые картинки с установить в качестве обоев.

# Tech Stack

- Jetpack Compose
- Accompanist
- MVVM
- Hilt
- Coil
- Retrofit
- Kotlin Serialization
